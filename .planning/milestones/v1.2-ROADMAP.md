# Milestone v1.2: Headless Login

**Status:** ✅ SHIPPED 2026-02-06
**Phases:** 8
**Total Plans:** 1

## Overview

Enable OAuth2 login on headless machines (EC2, SSH, containers) via RFC 8628 device authorization flow. Users can now authenticate with `gsuite login --no-browser` — a verification URL and code are printed to stderr, user authorizes on any device with a browser.

## Phases

### Phase 8: Device Authorization Flow

**Goal**: Add `--no-browser` flag to `gsuite login` using RFC 8628 device flow
**Depends on**: v1.1 complete
**Plans**: 1 plan
